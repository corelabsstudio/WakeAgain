1. 이 사이트는 무엇인가 (반드시 유지)
================================================================================

제품명: WakeAgain

한 줄:
  만들어 두고 홍보를 못 해서 잠든(방치된) 앱·프로그램·프로토타입을
  경매로 사고파는 마켓플레이스.

하는 일:
  - 판매자: 더 안 키울 사이드 프로젝트 / 팀 해산 후 남은 코드 / 서버만 꺼 둔 제품 등록
  - 구매자: 0부터 만들기 전에, 이미 손댄 베이스를 입찰로 이어받기
  - 핵심 UI: 현재 입찰가, 남은 시간, 입찰 버튼, 관심 등록, 프로젝트 올리기

플랫폼:
  웹 + Google Play + Apple App Store (같은 계정·데이터) — 디자인에는 "멀티 플랫폼" 정도만 암시

절대 바꾸지 말 것:
  - 브랜드 표기: WakeAgain (한글 음차 브랜드명 쓰지 말 것)
  - 비즈니스 모델 핵심: 잠든 디지털 제품의 경매·이전
  - 신뢰 필요: 거래/돈이 오가므로 사기·장난 사이트처럼 보이면 안 됨

2. 의도적으로 잊을 것 (기존 사이트 금지 목록)
================================================================================

아래는 이미 만든 홈의 틀이다. Variant에서는 이 틀을 따르지 말고 벗어난 안을 만들 것.

금지(기존 틀 재현 금지):
  - Soft dark 차콜 + 틸/민트 고정 팔레트 재사용
  - "기기 목업이 오른쪽에 있는 2컬럼 히어로" 고정
  - 상단 티커 + 카테고리 칩 + 3카드 그리드 + FAQ 아코디언 같은 기존 섹션 순서 복제
  - 보라색 AI 그라데이션, 흔한 "혁신 솔루션" SaaS 템플릿
  - 이모지 아이콘
  - 순흑(#000) 또는 순백(#FFF)만으로 된 피로한 풀스크린 (원하면 soft 변형은 OK)

기존 메타포 "버려진 PC, still running" 은:
  - 강제하지 말 것. 쓰고 싶으면 변형해도 되고, 완전히 다른 메타포여도 됨.
  - 단, "잠든 것이 다시 살아난다 / 다시 가치가 된다"는 감정은 유지.

3. 원하는 탐색 방향 (다양하게 뽑아 줄 것)
================================================================================

한 가지 정답이 아니라, 아래 방향마다 다른 비주얼 언어로 히어로 또는 풀 랜딩을 생성해 달라.

방향 A — 시네마틱 / 영화 오프닝
  풀스크린 입구, 큰 타이포, 입장 버튼, 드라마틱 여백, 스크롤 전 몰입

방향 B — 에디토리얼 / 매거진
  과감한 타이포 위계, 비대칭 레이아웃, 큰 헤드라인, 인쇄물 같은 리듬

방향 C — 라이브 마켓 / 거래소
  숫자·타이머·입찰 피드가 주인공, 긴장감 있지만 신뢰감 있는 금융·마켓 톤

방향 D — 미니멀 라이트
  밝고 청결, 넓은 여백, 선 하나·타입 하나로 고급스럽게 (눈 피로 적게)

방향 E — 웜 다크 / 나이트 스튜디오
  어둡지만 따뜻한 앰비언스, 부드러운 빛, 크리에이티브 툴 감성

방향 F — 스토리텔링 스크롤
  문제 → 장면 → 경매 → 신뢰 순으로 장면이 바뀌는 내러티브 랜딩

각 방향마다:
  - 색상 팔레트 3~5색
  - 히어로 레이아웃 스케치 수준으로 다르게
  - 경매 카드(입찰가+남은시간)는 반드시 한 번은 보이게

역동감:
  스크롤 등장, stagger, 타이머, 입장 버튼, hover 깊이
  정신없는 파티클/네온 파티는 피할 것 (거래 신뢰 유지)
