이용안내

잠든 프로젝트를 다시 깨우는 곳 — WakeAgain
쉽게 올리고, 쉽게 가격을 쓰되, 거래 규칙은 분명히 지킵니다.

WakeAgain(코어랩스)은 통신판매중개자입니다. 거래 당사자가 아니며, 매물 품질·사기 피해의 1차 책임은 판매자·구매자에게 있습니다. 운영 형식 검수·사이트 내 신용 점수는 보증이 아닙니다.

한 줄 흐름

  1. 올리기 — 데모·시작가 등록 → 형식 검수 후 공개 (등록 무료)
  2. 가격 쓰기 — 지금 가격·남은 시간 보고 입찰 (이메일 인증 Lv1이면 가능)
  3. 입금·이전 — 낙찰 후 안내에 따라 빠르게 입금 → 확인 후 자산 이전

팔 때 (판매자)

  1. 계정 가입 → 이메일 인증 → 실명·휴대폰(Lv2) → 판매자 공개 정보
  2. 프로젝트 올리기 (작동 확인·라이선스·처분 권한 체크)
  3. 운영 형식 검수 (보통 1~2영업일) 후 마켓에 공개
  4. 성사 시 판매자 수수료 10% · 입금 확인 전 코드·계정 이전 금지

살 때 (구매자)

  1. 마켓에서 지금 가격·남은 시간·데모를 확인
  2. 입찰은 Lv1 (이메일 인증) · 낙찰 후 결제·인수는 Lv2 (실명·휴대폰) · 즉시구매는 판매자가 선택으로 켠 매물만
  3. 낙찰되면 회사·판매자 안내에 따라 빠르게 입금 (1시간 자동 타이머는 PG 연동 후)
  4. 미입금 시 낙찰 무효·다음 순위 전환 가능 · 사이트 내 신용 점수 반영 가능

신원 단계 (Lv)

Lv0 가입이메일·비밀번호 · 공개 매물 열람 · 관심 등록
Lv1 이메일이메일 인증 · 입찰 · 관심
Lv2 신원실명·휴대폰 · 매물 등록 · 낙찰 후 결제·인수
Lv3 정산 계좌예금주·은행·계좌 · 판매자 정산·성사 확정

즉시구매 (선택)

  1. 필수가 아닙니다. 판매자가 비우면 경매(가격 쓰기)만 진행됩니다.
  2. 넣을 때만: 시작가 이상 ~ 최대 1억 원. 구매자가 그 가격에 바로 성사할 수 있습니다.
  3. 더 높은 입찰을 기다리려면 즉시구매를 두지 않는 편이 유리할 수 있습니다.
  4. 비현실적인 거액(999… 등)은 등록되지 않습니다.
