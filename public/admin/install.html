관리자 앱 설치

공개 사용자 앱과 완전히 분리된 운영용 홈 화면 앱입니다. 관리자 키가 맞을 때만 설치 UI가 열립니다. (Play/App 스토어 APK가 아니라 홈 화면 PWA · 카카오톡 안에서는 설치 불가)

1. 관리자 키 확인

서버 ADMIN_SECRET 과 같은 키를 입력하세요. 키는 이 기기에만 저장됩니다.

키를 입력한 뒤 확인을 누르세요.

2. 홈 화면에 관리자 앱 추가

키가 확인되면 아래 버튼으로 설치합니다. 설치 후 아이콘 이름은 WA Admin 입니다.

검수 화면 열기
  1. Android Chrome — 설치 버튼 또는 메뉴(⋮) → 앱 설치 / 홈 화면에 추가
  2. iPhone — Safari로 이 페이지 → 공유 → 홈 화면에 추가
  3. 카카오톡 인앱 — 메뉴 → 다른 브라우저로 연 뒤 설치

보안 안내

· 이 주소(/admin/install.html)는 사이트 메뉴에 노출하지 않습니다.
· 설치 자체는 “파일 잠금 다운로드”가 아니라, 키 없는 사람은 API·검수를 쓸 수 없음으로 보호합니다.
· 폰을 잃어버리면 홈 화면 앱 삭제 + ADMIN_SECRET 교체를 권장합니다.
